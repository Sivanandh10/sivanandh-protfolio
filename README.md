# Sivanandh C C | Royal Tech Portfolio 2.0

This portfolio is meticulously engineered to reflect the "Royal Tech" aesthetic suitable for a high-end Cloud Managed Services specialist at **Xebia**. It prioritizes architectural thinking, aesthetic restraint, and high-performance interaction mastery.

## 🏛️ Strategic Framework
- **Palette**: Midnight Opulence (#0B0F14, #1F2933, #C9A227).
- **Typography**: Playfair Display (Heritage Serif) & Inter (Modern Precision).
- **Interaction**: GSAP 3.0 (ScrollTrigger, SplitType, Horizontal Breakouts).
- **Background**: Three.js Procedural Particle Cloud.

## 🚀 Key Features
- **Inertial Motion**: Custom smooth-scroll feel with GSAP-driven section reveals.
- **Bento Logic**: Experience and skill domains organized in a luxury bento grid.
- **Horizontal Narrative**: Visual breakout sections for high-impact innovation statements.
- **Cloud Depth**: Project deep-dives focusing on "Architectural Decisions" and "Trade-offs".

## 🛠️ Tech Stack (Static High-Performance)
- **Frontend**: Vanilla HTML5 / CSS3 / ES6+
- **Animation**: GSAP (GreenSock Animation Platform)
- **3D Engine**: Three.js
- **Form**: EmailJS (Serverless)

## 📂 Project Structure
- `index.html`: Hero, Philosophy, Managed Domains, Horizontal Gallery.
- `projects.html`: Deep-dive architectural case studies (Theater Booking, ESEC Verify, etc.).
- `experience.html`: Vertical timeline of professional trajectory (Xebia, IBM, etc.).
- `contact.html`: High-end inquiry form.
- `js/animations.js`: Core GSAP orchestration logic.
- `js/background.js`: Three.js cloud particle engine.
- `css/style.css`: The "Royal Tech" design system.

## 🌐 Deployment
This is a high-performance static site. It can be deployed instantly via:
1. **GitHub Pages**: Upload the contents and enable Pages in settings.
2. **Vercel / Netlify**: Drag and drop the folder.
